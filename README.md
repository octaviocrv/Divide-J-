## Sobre o Projeto

O **Divide Já** nasceu de uma necessidade real que surgiu durante os intervalos e saídas com os amigos na PUC. Sempre que a conta da lanchonete ou do bar chegava, a matemática da divisão gerava confusão e perda de tempo. Para resolver esse problema clássico de forma definitiva, esta solução foi idealizada.

O projeto é uma aplicação web focada em utilidade e clareza, permitindo que os usuários gerenciem seu círculo de amigos, abram mesas virtuais, adicionem os itens consumidos e deixem que o sistema calcule automaticamente a fatia de cada participante na conta final. Uma abordagem prática e transparente para o dia a dia.

---

## Principais Funcionalidades

- **Dashboard Central:** Visão geral imediata com o resumo das mesas ativas, quantidade de amigos cadastrados e o montante financeiro total já dividido.
- **Gestão de Mesas:** Sistema intuitivo para criação de novas mesas, permitindo a adição de itens específicos e a marcação exata de quem participou de cada pedido.
- **Rede de Contatos:** Gerenciamento da lista de amigos dentro da plataforma, agilizando a inclusão de pessoas nas divisões de conta.
- **Histórico e Transparência:** Registro completo de todas as mesas encerradas recentemente para consultas futuras e prestação de contas.
- **Perfil Exclusivo:** Sistema de usuário customizado com tag única de identificação (ex: #3345) e um painel de configurações para ajustes de conta.

## Tecnologias Utilizadas

- **React & Next.js** — Estruturação completa da interface de usuário, garantindo uma navegação rápida e roteamento eficiente.
- **TypeScript** — Tipagem estática aplicada para garantir maior segurança, escalabilidade e previsibilidade no código do projeto.
- **Tailwind CSS** — Estilização utilitária focada na construção de um design limpo, moderno e totalmente responsivo para dispositivos móveis e desktops.
- **Node.js** — Ambiente de execução base para gerenciamento de dependências e integração do ecossistema de desenvolvimento.
